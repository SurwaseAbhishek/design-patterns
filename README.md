# Design Patterns in Java

This repository contains implementations of various **design patterns** in Java, demonstrating best practices in object-oriented design.

## 📁 Project Structure

This project is structured as a **Maven-based Java application** using Eclipse IDE.

### 🧱 Categories Covered:

- **Creational Patterns**
  - Singleton
  - Factory
  - Abstract Factory
  - Builder
  - Prototype

- **Structural Patterns**
  - Adapter
  - Decorator
  - Proxy
  - Composite
  - Facade
  - Bridge
  - Flyweight

- **Behavioral Patterns**
  - Strategy
  - Observer
  - Command
  - Chain of Responsibility
  - Template Method
  - State
  - Iterator
  - Mediator
  - Memento

---

## 🔧 Prerequisites

- Java 8 or higher
- Maven 3.x
- Eclipse IDE (or any Java IDE)
